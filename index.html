Credit Risk Analysis Dashboard

Basel II / III Framework · 32,581 Loans · Python / Plotly Dash

Overview

A CRO-level credit risk platform built on a 32,581-loan consumer dataset. The dashboard implements Basel II/III methodology — computing PD scores by loan grade, LGD (45% unsecured), and EAD to derive Expected Loss at portfolio and segment level. Features include a WOE/IV PD scorecard, interactive stress testing (PD multiplier, LGD, EAD growth), and a fully filterable loan explorer with drill-down to individual records.
